# ToyTagged

Aplicación web desarrollada en PHP y MySQL que permite a los usuarios gestionar publicaciones de colecciones de juguetes mediante imágenes, descripciones y etiquetas.


## Funcionalidades

- Registro de usuarios
- Inicio de sesión
- CRUD de publicaciones
- Subida de imágenes
- Sistema de etiquetas
- Filtrado de publicaciones
- Panel de usuario
- Control de permisos


## Tecnologías utilizadas

- PHP
- MySQL
- HTML
- Tailwind CSS
- XAMPP
- phpMyAdmin


## Requisitos

Para ejecutar el proyecto es necesario disponer de:

- XAMPP o entorno similar
- PHP 8 o superior
- MySQL
- Navegador web


## Instalación

1. Copiar la carpeta del proyecto dentro de la carpeta htdocs
2. Iniciar Apache y MySQL desde XAMPP
3. Importar la base de datos desde phpMyAdmin
4. Acceder desde el navegador:

https://github.com/Tomas-Ca/TFG
